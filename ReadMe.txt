Code consists of 6 java file (5 clients & 1 server)
Build each of the java files

Run server class 
java server <FilePath> <server port#> <client1 port#> <client2 port#> <client3 port#> <client4 port#> <client5 port#>

Run all clients parallely
java client1 <DestinationFolder> <server port#>
java client2 <DestinationFolder> <server port#>
java client3 <DestinationFolder> <server port#>
java client4 <DestinationFolder> <server port#>
java client5 <DestinationFolder> <server port#>


File will uploaded to each client
